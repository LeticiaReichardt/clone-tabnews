# clone-tabnews

Projeto ponto a ponto, relembrando tecnologias.

Pontos fortes do projeto:

- Utilizando testes automatizados para então criar as funções necessarias, com a logica que precisa-se chegar no resultado x programado no teste e a criação da função e testada até o resultado x ser alcançado.

- Fremeworks para padronização de alinhamento de código, para ajudar quando o desenvolvimento e feito por uma equipe. Realiza os ajustes no código de forma automatica.

- Utilização de API para acesso ao banco de dados.

- Padrão de testes no estilo estrela.
